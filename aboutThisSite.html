About this site

This portfolio started with Udacity's course "Intro to HTML and CSS".

The technologies used to build this site was HTML5, Javascript, CSS, Bootstrap and JQuery frameworks.

Perfomance was enhanced with minification of scripts, compressed images, use of CDNs and with the help of Google's tool PageSpeed Insights

Responsiveness was a priority to the construction of this website, since it is important to a website to work in many devices, from a small phone to a big desktop screen.

But keep in mind that this website is still under construction and it's going to get much better.
If you have any suggestion, you can write it out in the contact page.
Or, if you would like to contribute to the source code, feel free to go to my github.

Thanks for visiting!
