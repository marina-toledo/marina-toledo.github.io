Resume project

This project was built in the Udacity's "Javascript Basics" course.
It is an interactive curriculum built with Javascript. It gets the contents of your resume from a JSON file and dynamically displays the content within a provided template.
Technologies used are HTML5, CSS, Javascript and Jquery framework.

Your moving companion

This project was built in the Udacity's "Intro to AJAX" course.
It is a single-page application in which you can look up for places, giving the street and city address. And then the browser respond with a street view of the place, with New York Times articles and with Wikipedia's articles about that location.
Technologies used are HTML, CSS, Javascript and Jquery framework.
